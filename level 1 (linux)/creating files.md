#File Creation using command line

[touch filename.txt] -  this code creates a new file called filename in your pwd

[echo "Content" > file.txt] - this creates a new file called file and adds the words inside "" as the content of the file

[echo "Content" > .hiddenfile] - this command creates a hidden file called hiddenfile and adds the words inside the "" as the content of the file

##note - hidden files are created if file name is written after {.} and wont appear on the screen under normal listing circumstances
