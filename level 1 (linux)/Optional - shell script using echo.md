`echo '#!/bin/bash\necho "HELLO"' > script.sh` 
- `echo '#!/bin/bash` - Tells the system it is a bash script and to call the bash interpreter
- `\n` - new line to make sure the both are in different lines
- `echo "HELLO"` - content that is stored in th script.sh, prints `HELLO` everytime the script is executed
- `>` - points towards script.sh
- `script.sh` - shell script - it contains an sequentially executable series of commands
