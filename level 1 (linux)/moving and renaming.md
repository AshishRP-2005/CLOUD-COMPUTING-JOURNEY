#MOVING

[mv file.txt newname.txt] - mv command is used to change the file name to the specfied new name

[mv file.txt directory1/] - moves the file from the directory it is present in to the specified directory

[mv file.txt directory1/newname.txt] - moves file to the specified directory and renames it to the specified name {here it is newname.txt}

[mv directory1 directory2] - renames the specified directory to the new name

[mv testdirectory/newname.txt  ./original.txt] - moves the file newname.txts out of test directory and renames it as original.txt
